🎯 Quiz Not Being Answered? Use Iframe Method!

If the Next button works but quizzes aren't being answered, the quiz is in a cross-origin iframe. Follow these steps to run the script INSIDE the iframe:

1 Open Browser Console

Press F12 to open Developer Tools

2 Switch to H5P Iframe Context

At the top of the Console tab, you'll see a dropdown that says top

Click it and select the iframe that contains "h5p_embed" in the URL

3 Copy the Iframe Script

Click this button to copy:

4 Paste and Run

Paste the script in the console (make sure you're in the iframe context!) and press Enter

5 See It Work!

You should see:

🎯 LMS Quiz Helper (Iframe Mode) Started!
ℹ️ Monitoring for quizzes...
🎯 Quiz detected!
ℹ️ Attempt 1
✅ Correct!
✅ Clicked continue
⚠️ Important: You need to do this for EACH video/topic. When you click Next and go to a new video, the iframe reloads, so you'll need to switch to the iframe context and paste the script again.

💡 Why This Happens

The quiz is inside an iframe from a different domain (content.yourhippo.com) than the main page (lms.yourhippo.com). Browsers block scripts from accessing cross-origin iframes for security.

By running the script INSIDE the iframe, it has direct access to the quiz elements.
